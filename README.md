# RAG Enterprise CV Analysis System for Germany

A comprehensive Retrieval-Augmented Generation (RAG) system for analyzing German CVs using LLMs, Neural Networks, and advanced NLP techniques.

## Features

- **PDF/DOCX/TXT CV Parsing**: Intelligent extraction of structured data from German CVs
- **Semantic Search**: FAISS-powered vector search with multilingual embeddings
- **LLM Integration**: OpenAI API integration for intelligent CV analysis
- **Neural Network Classification**: Deep learning-based CV categorization
- **REST API**: Enterprise-ready API endpoints for system integration
- **Streamlit UI**: Interactive web interface for easy deployment

## Technology Stack

- **RAG Framework**: FAISS + Sentence Transformers
- **LLM**: OpenAI GPT-3.5/GPT-4
- **Neural Networks**: PyTorch
- **Embeddings**: Multilingual Sentence-BERT
- **UI**: Streamlit
- **Document Processing**: PyPDF2, python-docx
